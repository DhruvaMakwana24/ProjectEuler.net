## 🎯 Core Objectives
This repository tracks my solutions for Project Euler. My primary goals with this project are:
- **Mathematical & Statistical Proficiency:** Deepening my understanding of number theory, progressions (AP/GP), combinatorics, and statistical patterns.
- **Performance Engineering:** Abiding by Project Euler's "one-minute rule" by writing efficient algorithms ($O(1)$ or $O(\log n)$ instead of brute-force $O(n^2)$ loops).
- **Modern Architecture:** Applying test-driven development (xUnit) and performance benchmarking (BenchmarkDotNet) to mathematical solutions.
